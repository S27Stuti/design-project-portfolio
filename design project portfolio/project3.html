PROJECT-3:-Travelling Website – UI/UX Design

Introduction

This project is a Travelling Website UI/UX design created to help users easily plan their trips.

The website allows users to book flights, explore attractions in different countries, rent cars, and manage airport travel services.

The design focuses on simple navigation, quick booking, and a visually attractive travel experience.

Project Objective

  • To design a user-friendly travel booking website
  • To simplify flight and hotel booking
  • To showcase popular tourist attractions
  • To provide transport and airport travel services

Problem Statement

  • Complex travel booking websites
  • Confusing navigation for users
  • Too many steps to book flights and hotels
  • Lack of clear travel information

Proposed Solution

  • Clean and simple UI layout
  • Step-by-step booking process
  • Clear navigation menu
  • Dedicated sections for attractions and transport

Tools Used

  • Figma – UI/UX design and prototyping
  • Icons & UI components
  • Color theory & typography

Design Approach

  • User-centered design
  • Minimal and modern UI
  • Consistent colors and fonts
  • Easy-to-use booking flow

Screen-Wise Explanation

1. Homepage

  • Logo and branding
  • Navigation menu for Flights, Hotels, Attractions
  • Search section and featured travel deals

2. Login Screen

  • Email and password login
  • Forgot password option

3. Create Account

  • User registration details
  • Sign up button

4. Account Verification

  • OTP verification
  • Email or phone confirmation

5. Flight Booking

  • From / To locations
  • Date and passenger selection

6. Attractions by Country

  • India, USA, UK, Canada, Australia, Africa
  • Popular tourist places

7. Airport Travel

  • Pickup and drop services
  • Passenger selection

8. Taxi & Car Rental

  • Taxi, SUV, luxury car options
  • Hourly and daily rentals

9. Flight + Hotel Package

  • Combined travel deals
  • Easy booking options

What I Learned

I learned how to design a complete travel booking website, manage complex user flows, and create a smooth booking experience.

Conclusion

This project demonstrates my ability to design a professional and user-friendly travel website with end-to-end UI/UX flow.
